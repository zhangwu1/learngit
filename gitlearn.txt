1、安装Git只需要按照默认安装即可，自己需要对安装目录进行修改，需要确认的是安装目录不可以存在中文，以防后期产生一系列问题
2、创建版本库并上传文件
创建一个空目录初始化Git仓库：git init
创建文件：touch 文件名 或者直接使用vi 文件名
将创建的文件提交到版本库：首先"git add 文件名"  然后再通过git commit -m "提交备注信息"
git add 文件时可以一次性提交多个文件，例如git add file1 file2 file3
git commit 命令会将提交的文件均上传至版本管理库中
3、版本回退
每次的提交记录均可以通过"git log"命令查看具体的信息
如果只想展示一行数据可以通过"git log --pretty=oneline"查看历史信息
每次提交均对应一个版本号，可以通过版本号对历史版本进行回退
如果想将版本退回到上一个版本，可以通过git reset --hard 版本号（可以只写前几位）
退回版本过程中可能会出现退回到上一个版本文件被删除了，此时需要通过未关闭的窗口中找到退回前的版本号，在通过"git reset --hard 版本号"可以找回文件，很显然也曾存在另外一种情况，我们关闭了当前窗口，也可以通过"git reflog"查看所有的历史记录。
